Les TPs Technologie du Web



 Ce site concerne ce que nous avons étudié dans cet module "Technologie du Web".
                    
Introduction:


                    La technologie du Web est un ensemble de technologies qui permettent de créer et de publier des contenus en ligne,

        ainsi que de les partager et de les consulter sur Internet. 

        Le Web est constitué de millions de sites Web qui sont connectés entre eux et qui sont accessibles à tous à l'aide d'un navigateur Web.



                    Pour créer et publier des sites Web, on utilise des langages de programmation tels que HTML (HyperText Markup Language),

        CSS (Cascading Style Sheets) et JavaScript.

        HTML sert à structurer le contenu du site, CSS sert à lui donner du style et JavaScript sert à ajouter de la interactivité et des fonctionnalités supplémentaires.



                    Le Web est en constante évolution et de nouvelles technologies sont régulièrement introduites pour améliorer l'expérience utilisateur et ajouter de nouvelles 

        fonctionnalités aux sites Web. Parmi ces technologies, on peut citer les réseaux sociaux, les applications mobiles, les APIs (Application Programming Interfaces), 

        les données en temps réel, etc.



                    En tant qu'utilisateur du Web, vous pouvez accéder à tous ces contenus en ligne en utilisant un navigateur Web sur votre ordinateur,
 
        votre smartphone ou votre tablette. Vous pouvez également utiliser des moteurs de recherche pour trouver des informations sur le Web et des services en ligne pour 

        effectuer des actions comme acheter en ligne, envoyer des emails, partager des photos, etc.

                    
Definition de HTML, CSS et JavaScript:

                    
HTML:

        
         
                    HTML (HyperText Markup Language) est un langage de programmation utilisé pour structurer et présenter le contenu d'un site Web.

            HTML permet de définir la structure du document, comme les titres, les paragraphes, les listes, les tableaux, etc. 
        

                    
CSS:

        
         
                    CSS (Cascading Style Sheets) est un langage de programmation utilisé pour définir l'apparence et le style d'un site Web. 

            CSS permet de définir la mise en forme du document, comme les couleurs, les polices, les marges, les bordures, etc.
        

                    
JavaScript:

        

                    JavaScript est un langage de programmation de scripts côté client utilisé pour ajouter de l'interactivité et des fonctionnalités supplémentaires aux sites Web.

            JavaScript permet de créer des effets visuels, de valider des formulaires, de créer des animations, etc.
